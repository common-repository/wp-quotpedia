Wordpress için Quotpedia sözleri.
 
== Description ==
 
Wordpress için hazırlanmış olan bu eklenti ile bloğunuza Quotpedia sözleri ekleyebilirsiniz.

WP-Quotpedia, Quotpedia sitesinde yayınlanan söz, atasözü, özlü sözler, güzel sözler, günün sözü, replik, gaf, kitap ve şiirlerde geçen sözleri sitenizde yayınlayabilmenize imkan sunar.

Eklentiyi yükleyip, aktifleştirdikten sonra; yönetim panelinizin "Bileşenler" sayfasından temanızın bileşen desteği olan alanlara [Quotpedia](http:/www.quotpedia.com/) sözlerini yerleştirebilrisiniz.

== Frequently Asked Questions ==

= WP-Quotpedia nedir? =

WP-Quotpedia eklentisi, [Quotpedia](http:/www.quotpedia.com/) sitesinde yayınlanan sözleri kendi sitenize taşımanıza imkan sunar.

= WP-Quotpedia hangi dilleri destekler? =

WP-Quotpedia, şimdilik [Quotpedia Türkçe](http://tr.quotpedia.com/) için kullanılabilmektedir. Çok yakında [Quotpedia İngilizce](http://en.quotpedia.com/) için de destek verecektir.
